NLS Module Reference

The NLS (Nexus Ladder Scheduler) module is pre-installed in the sandbox and available to all workflow scripts. It provides access to the Nexus credential store and script library.

Set Server URL

Override the Nexus server URL. Defaults to http://localhost:8080 or the NEXUS_SERVER_URL environment variable. You typically don't need this inside the container.

PowerShell Python 
Set-NLSServer -Url 'http://nexus-app:8080'
import nls
nls.set_server("http://nexus-app:8080")

Get Credential

Retrieve credentials from the Nexus credential store with all secret fields decrypted.

PowerShell Python 
# List all credentials (metadata only)
Get-NLSCredential

# Get a specific credential (decrypted)
$creds = Get-NLSCredential -Name 'prod-db-login'
$creds.username   # 'dbadmin'
$creds.password   # 'supersecret'

# Return as PSCustomObject instead of hashtable
$creds = Get-NLSCredential -Name 'prod-db-login' -AsObject
ParameterTypeRequiredDescription
-NameStringNoCredential name. Omit to list all.
-AsObjectSwitchNoReturn PSCustomObject instead of hashtable.
import nls

# Get a specific credential (decrypted)
creds = nls.get_credential("prod-db-login")
print(creds["username"])   # 'dbadmin'
print(creds["password"])   # 'supersecret'

# Use with boto3
aws = nls.get_credential("aws-production")
import boto3
session = boto3.Session(
    aws_access_key_id=aws["accessKeyId"],
    aws_secret_access_key=aws["secretAccessKey"],
)
ParameterTypeRequiredDescription
namestrYesCredential name to retrieve.

List Scripts

List available scripts in the Nexus script store by type.

PowerShell Python 
# List all PowerShell scripts
Get-NLSScript -ScriptType PowerShell

# Filter results
Get-NLSScript -ScriptType Terraform | Where-Object { $_.Name -like '*deploy*' }
ParameterTypeRequiredDescription
-ScriptTypeStringYesPowerShell, Terraform, Python, or Shell

Returns objects with: Name, Size, LastModified

import nls

# List all PowerShell scripts
scripts = nls.get_scripts("powershell")
for s in scripts:
    print(f"{s['name']}  ({s['size']} bytes)")

# List terraform scripts
nls.get_scripts("terraform")
ParameterTypeRequiredDescription
script_typestrYes"powershell", "terraform", "python", or "shell"

Returns a list of dicts with keys: name, size, lastModified

Copy Script

Download a script from the Nexus script store to the sandbox workspace (or a custom directory).

PowerShell Python 
# Copy to default workspace
Copy-NLSScript -ScriptName 'createPostgresCopy.ps1' -ScriptType PowerShell

# Copy to a subdirectory
Copy-NLSScript -ScriptName 'main.tf' -ScriptType Terraform -Destination '/home/sandbox/workspace/infra'
ParameterTypeRequiredDescription
-ScriptNameStringYesFilename of the script to copy.
-ScriptTypeStringYesPowerShell, Terraform, Python, or Shell
-DestinationStringNoTarget directory. Default: /home/sandbox/workspace

Returns the full path of the copied file.

import nls

# Copy to default workspace
path = nls.copy_script("createPostgresCopy.ps1", "powershell")
print(f"Saved to {path}")

# Copy to a subdirectory
path = nls.copy_script("main.tf", "terraform", destination="/home/sandbox/workspace/infra")
ParameterTypeRequiredDescription
script_namestrYesFilename of the script to copy.
script_typestrYes"powershell", "terraform", "python", or "shell"
destinationstrNoTarget directory. Default: /home/sandbox/workspace

Returns the full path string of the copied file.

Quick Reference

PowerShell

Import-Module NLS               # auto-loaded in the sandbox

Set-NLSServer -Url '...'
Get-NLSCredential [-Name '...'] [-AsObject]
Get-NLSScript -ScriptType PowerShell|Terraform|Python|Shell
Copy-NLSScript -ScriptName '...' -ScriptType PowerShell|Terraform|Python|Shell [-Destination '...']

Python

import nls

nls.set_server("...")
creds = nls.get_credential("name")
scripts = nls.get_scripts("powershell")
path = nls.copy_script("file.ps1", "powershell")
